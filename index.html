08 / El plano

      
Mapa de arquitectura

      
Toda app tiene estas capas. Piénsalo como un restaurante: un salón (lo que ve el cliente), una cocina (procesa) y una despensa (guarda).

      

        
 FRONTEND — el salónLo que el usuario ve
Interfaz (UI)Botones, pantallas, colores
InteracciónQué pasa al hacer clic

        
▼

        
 BACKEND — la cocinaEl cerebro, invisible
LógicaLas reglas: "si X entonces Y"
APIEl mesero que lleva y trae
AutenticaciónQuién eres y si puedes entrar

        
▼

        
 BASE DE DATOS — la despensaDonde vive la info
Datos guardadosUsuarios, publicaciones...
ArchivosFotos, documentos

        
▼

        
 DEPLOY — abrir el restaurantePublicar para otros
HostingEl servidor donde vive online
DominioTu dirección: mi-app.com

      

      
No toda app necesita todas las capas. Una calculadora solo necesita frontend; una red social, las cuatro.
