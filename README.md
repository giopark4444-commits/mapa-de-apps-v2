# Mapa de Apps · Tu copiloto de vibecoding

App de un solo archivo (`index.html`) que ayuda a personas con ideas —pero sin
conocimientos de código— a planear, construir y publicar una app con la ayuda de
Claude Code. Se abre con doble clic en cualquier navegador; no requiere instalar
nada y todo se guarda en tu propio navegador.

## Qué incluye

- **Mis proyectos** — gestiona varias ideas a la vez, cada una con su avance.
- **Proyecto actual** — describe tu idea y obtén un resumen técnico, un ranking de
  tecnologías con el porqué, costos estimados y el prompt perfecto para Claude.
- **Roadmap del proyecto** — las fases del ciclo de vida, adaptadas a tu tipo de app.
- **Llevar a dispositivos** — checklist para iPhone, Android, tablet y web.
- **Analizar una app** — pega un repo de GitHub y audita qué tiene, qué le falta y
  cómo solucionarlo (con prompts), incluyendo compatibilidad por dispositivo.
- **Diccionario, prompts y errores comunes** — guía de referencia sin tecnicismos.

## Uso

Abre `index.html` en tu navegador. Listo.
